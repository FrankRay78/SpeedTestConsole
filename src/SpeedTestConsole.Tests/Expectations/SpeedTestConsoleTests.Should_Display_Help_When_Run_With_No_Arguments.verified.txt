DESCRIPTION:
Internet speed tester including server discovery, latency measurement, and download speed testing.

USAGE:
    SpeedTestConsole [OPTIONS] <COMMAND>

OPTIONS:
    -h, --help    Prints help information

COMMANDS:
    servers     Show the nearest speed test servers    
    download    Perform an internet download speed test
